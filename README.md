AI 면접관을 통해 맞춤형 면접 질문과 피드백을 제공하는 웹서비스
# My AI Interviewer

## 프로젝트 소개
사용자의 경험과 관심 직무를 기반으로
AI 면접관이 맞춤형 질문과 피드백을 제공하는 웹서비스입니다.

## 주요 기능
- 사용자 정보 입력
- 관심 직무 선택
- AI 면접 질문 생성
- 답변 피드백 제공
- 면접 결과 리포트 확인

## 제작 목적
취업 준비 과정에서 반복적인 면접 연습의 한계를 해결하고,
개인 경험에 맞춘 면접 연습 환경을 제공하기 위해 제작했습니다.

## 사용 기술
- HTML
- CSS
- JavaScript
- Google Antigravity
